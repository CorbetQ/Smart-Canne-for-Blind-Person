# Smart Canne
## Smart stick for blind person

### Inspired by Third Eye project by RobotTechMaker and my girlfriend
### https://www.instructables.com/THIRD-EYE-FOR-BLINDS-an-Innovative-Wearable-Techno/

#### Input Voltage : 9V DC
#### Used Voltage  : 5V DC
#### Current       : 100 mA
#### Power         : 500 mW
#### Alert Voltage : 4.55V
#### Stop Voltage  : 4.4V
#### TRG PIN       : D5 PWM
#### Echo PIN      : D6 PWM
#### Buzzer + PIN  : D3 PWM
#### Voltage Tester: AO
#### Stick size    : 1,20m
